Сreates a primitive data structure.

Parameters:

  • vertices table Table or Data of vertices
  • indices table Table or Data of indices
  • count number Count of vertices
  • Image image Image object that is used when drawing the mesh
  • vertexformat table A table in the form of {attribute, ...}_
  • color table Сolor table that is used when drawing the mesh
  • indices_type_size number Size of the index type if the indices stored in Data [2 or 4]
