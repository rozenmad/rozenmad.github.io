Module core3d.environment

An environment is a class that sends information about the current settings of the environment (such as ambient color, fog, light sources, camera transformation matrices) etc to the shader.

send_uniforms_to (shader)
Sends all the environment uniforms to the shader. This function can be used when creating your own display objects, or for shading technique. This method is called automatically when the environment is used in scene:render_nodes()

apply_shader (shader)
Sets a Shader as the current pixel effect or vertex shaders. All drawing operations until the next apply will be drawn using the Shader object specified. This method is called automatically when the environment is used in scene:render_nodes()
